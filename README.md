## @kalutskii/framework

Lightweight TypeScript toolkit for creating consistent, reusable, and cross-service protocol layers:

- Unified patterns for service communication  
- Modular namespaces for clean separation of concerns  
- Reusable protocol primitives across distributed applications  
- Environment-agnostic (Bun, Node, Workers)

---

### Installation

```bash
bun add @kalutskii/framework@github:kalutskii/framework
# or
pnpm add @kalutskii/framework@github:kalutskii/framework
# or
npm add @kalutskii/framework@github:kalutskii/framework
